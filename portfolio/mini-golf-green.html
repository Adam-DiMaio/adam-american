Mini Electronic Golf Green

Custom Sensors & Automated Gameplay

Project Overview

This project simulates a mini electronic golf green with custom sensors and programming to mimic real golf green conditions, providing an interactive experience.

Key Contributions & Responsibilities

  • Designed & implemented custom sensor circuits for environmental monitoring.
  • Developed programming logic to automate gameplay dynamics.
  • Contributed to physical layout & modeling of the golf green simulation.

Skills & Technologies

  • Electronics: Custom sensor circuit design & integration
  • Programming: Embedded programming for automation
  • Design: Physical layout & modeling

Outcome

The project successfully created an interactive simulation that mimics real-world golf green conditions, providing a unique user experience.

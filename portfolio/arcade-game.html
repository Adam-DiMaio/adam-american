Arcade Game

Retro-Style Gameplay & Interactive Design

Project Overview

This project involved designing and developing a retro-style arcade game by integrating microcontrollers with interactive design concepts.

Key Contributions & Responsibilities

  • Collaborated on the game design process, blending hardware & software.
  • Implemented embedded programming to drive game mechanics.
  • Utilized 3D printing & laser cutting for physical components.

Skills & Technologies

  • Microcontrollers: Embedded programming & circuit integration
  • Game Design: Conceptualizing interactive mechanics

Outcome

The project resulted in a fully functional, interactive arcade game that effectively showcased the integration of digital and physical design elements.

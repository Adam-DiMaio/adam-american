R&D Scholar: Proprietary Wearable Medical Device

Utah Tech University · INSPIRE Program

Project Overview

This project was part of the INSPIRE program at Utah Tech University. I collaborated with a multidisciplinary team to develop a wearable medical device designed to monitor real-time health parameters for knee reconstruction patients.

Key Contributions & Responsibilities

  • Collaborated on sensor integration and ergonomic design.
  • Utilized CAD tools (Onshape, SolidWorks) for device component design.
  • Participated in prototyping and testing to ensure optimal functionality.

Skills & Technologies

  • Medical Engineering: Advanced sensor integration and patient-centric design.
  • Wearable Design: Emphasis on ergonomics and usability.
  • CAD & Prototyping: Iterative design using Onshape and SolidWorks.

Outcome

The project successfully demonstrated a functional prototype that improved patient recovery outcomes and provided valuable interdisciplinary experience.
